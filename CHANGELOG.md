# 更新日志

本文件用于记录此项目每次可见功能更新。后续每次修改站点时，都会同步追加一条记录。

## 2026-04-05

### v1 初版上线

- 建立手机可访问的川藏 ADV 摩旅路书站点
- 增加分日路书、装备清单、应急预案、出发检查
- 增加海拔剖面图和路线切换
- 增加骑行模式 UI
- 增加 GPX 导入
- 增加 GitHub Pages 在线访问

### v1.1 PWA 与分享优化

- 增加 `manifest.webmanifest`
- 增加 `pwa-icon.svg`
- 增加首页分享文案
- 增加安装到主屏幕入口

### v1.2 稳定性修复

- 修复 GitHub Pages 浏览器运行时启动问题
- 修复页面偶发“看得到但点不动”的问题
- 调整缓存策略，避免旧版本反复回流
- 增加自动清理旧 `service worker` 和旧缓存逻辑

### v1.3 内容增强

- 增加“经验”页签
- 增加出发窗口、骑行节奏、通讯与导航、住宿判断、装备取舍、安全底线
- 增加“费用与补给”模块
- 增加“每日图片 / 实景卡片”模块

### v1.5 日志机制
