# 川藏线 ADV 摩旅路书

**适用车型：** ADV / 拉力车（排量 400cc 以上推荐）  
**组队规模：** 3–4 人  
**总里程：** 约 2,150 km（318 国道成都→拉萨）  
**建议总天数：** 12–14 天（含休整日）  
**最佳窗口：** 6 月中旬 – 10 月上旬（避开雨季高峰 7 月下旬 – 8 月中旬为宜）

---

## 第一部分 · 分日路书

> 每日控制在 200–350 km 以内；海拔跨度大的路段缩短里程，留足适应时间。

### Day 1 ｜ 成都 → 泸定（约 280 km）

| 项目 | 说明 |
|------|------|
| 路况 | 成雅高速 + G318，柏油路面为主，雅安后开始盘山 |
| 海拔 | 500 m → 1,330 m |
| 关键节点 | 雅安（补给站）→ 天全 → 二郎山隧道（4,176 m 隧道，注意温差）→ 泸定 |
| 加油 | 雅安、天全各有中石油/中石化 |
| 住宿建议 | 泸定县城宾馆，条件尚可 |
| 注意事项 | 二郎山隧道内湿滑，降低车速；出隧道后温差大，注意起雾 |

### Day 2 ｜ 泸定 → 新都桥（约 170 km）

| 项目 | 说明 |
|------|------|
| 路况 | G318 盘山路，康定后翻越折多山，弯道密集 |
| 海拔 | 1,330 m → 折多山垭口 4,298 m → 新都桥 3,460 m |
| 关键节点 | 泸定桥（可短暂参观）→ 康定（最后大型补给点）→ 折多山垭口 → 新都桥 |
| 加油 | 康定城区加满，新都桥有加油站但偶尔缺油 |
| 住宿建议 | 新都桥镇藏式客栈或酒店 |
| 注意事项 | 折多山是全程第一个高海拔垭口，注意高反初期症状；下山弯道多，湿滑路段保持低档位 |

### Day 3 ｜ 新都桥 → 雅江 → 理塘（约 210 km）

| 项目 | 说明 |
|------|------|
| 路况 | 翻越高尔寺山、剪子弯山、卡子拉山，连续高海拔 |
| 海拔 | 3,460 m → 卡子拉山 4,718 m → 理塘 4,014 m |
| 关键节点 | 高尔寺山隧道 → 雅江（午餐补给）→ 剪子弯山 → 卡子拉山 → 理塘 |
| 加油 | 雅江、理塘均有加油站 |
| 住宿建议 | 理塘县城酒店（世界高城，注意保暖） |
| 注意事项 | 连续三个 4,000 m+ 垭口，头痛、胸闷属正常反应；理塘海拔高、夜间温度可降至 0°C 以下，务必备好保暖层 |

### Day 4 ｜ 理塘 → 巴塘（约 170 km）

| 项目 | 说明 |
|------|------|
| 路况 | 海子山垭口后长下坡，路况良好 |
| 海拔 | 4,014 m → 海子山 4,685 m → 巴塘 2,580 m |
| 关键节点 | 海子山（姊妹湖观景台）→ 巴塘 |
| 加油 | 巴塘加油站 |
| 住宿建议 | 巴塘县城 |
| 注意事项 | 海子山到巴塘落差超 2,000 m，长下坡注意刹车温度；巴塘海拔较低，适合高反恢复 |

### Day 5 ｜ 巴塘 → 芒康 → 左贡（约 260 km）

| 项目 | 说明 |
|------|------|
| 路况 | 跨金沙江进入西藏，翻越宗巴拉山、拉乌山、觉巴山、东达山 |
| 海拔 | 2,580 m → 东达山 5,130 m → 左贡 3,877 m |
| 关键节点 | 金沙江大桥（川藏界）→ 芒康（检查站，备好身份证）→ 觉巴山 → 东达山垭口 → 左贡 |
| 加油 | 芒康、左贡 |
| 住宿建议 | 左贡县城 |
| 注意事项 | 东达山是 318 线最高垭口之一（5,130 m），可能有积雪或暗冰；芒康检查站需登记，保持证件随身 |

### Day 6 ｜ 左贡 → 八宿（约 200 km）

| 项目 | 说明 |
|------|------|
| 路况 | 邦达草原 → 业拉山（怒江 72 拐），全程最险路段之一 |
| 海拔 | 3,877 m → 业拉山 4,618 m → 怒江河谷 2,740 m → 八宿 3,260 m |
| 关键节点 | 邦达 → 业拉山 → 怒江 72 拐（约 30 km 连续发卡弯） → 八宿 |
| 加油 | 邦达、八宿 |
| 住宿建议 | 八宿县城 |
| 注意事项 | 72 拐是全程技术难度最高路段：急弯 + 落差 + 可能碎石路面；保持低速、靠内侧、注意对向大车；建议队伍前后车保持无线电联络 |

### Day 7 ｜ 八宿 → 然乌 → 波密（约 220 km）

| 项目 | 说明 |
|------|------|
| 路况 | 安久拉山后进入林区，路况转好 |
| 海拔 | 3,260 m → 安久拉山 4,468 m → 然乌湖 3,850 m → 波密 2,750 m |
| 关键节点 | 安久拉山 → 然乌湖（建议停留拍照）→ 米堆冰川（可选往返约 2 小时）→ 波密 |
| 加油 | 然乌、波密 |
| 住宿建议 | 波密县城，条件较好 |
| 注意事项 | 然乌到波密路段雨季可能塌方或泥石流，提前查看路况信息 |

### Day 8 ｜ 波密 → 通麦 → 鲁朗 → 林芝（约 230 km）

| 项目 | 说明 |
|------|------|
| 路况 | 通麦天险已改为隧道+桥梁，大幅改善；翻越色季拉山 |
| 海拔 | 2,750 m → 色季拉山 4,728 m → 林芝 2,930 m |
| 关键节点 | 通麦特大桥 → 鲁朗（石锅鸡午餐推荐）→ 色季拉山（天气好可远眺南迦巴瓦）→ 林芝 |
| 加油 | 鲁朗、林芝 |
| 住宿建议 | 林芝八一镇，城市级住宿 |
| 注意事项 | 色季拉山常年云雾大，能见度可能极低；鲁朗林海路段弯道较多但路面好 |

### Day 9 ｜ 林芝 休整日

| 项目 | 说明 |
|------|------|
| 安排 | 车辆全面检查（机油、刹车片、链条/传动、轮胎气压）；补充物资；高反恢复 |
| 可选 | 雅鲁藏布大峡谷一日游、南迦巴瓦观景台 |

### Day 10 ｜ 林芝 → 工布江达（约 270 km）

| 项目 | 说明 |
|------|------|
| 路况 | 林拉公路，路况极佳，部分高速化 |
| 海拔 | 2,930 m → 工布江达 3,440 m |
| 关键节点 | 尼洋河风光带 → 工布江达 |
| 加油 | 沿途加油站充足 |
| 住宿建议 | 工布江达县城 |
| 注意事项 | 路况好容易超速，注意测速和横风 |

### Day 11 ｜ 工布江达 → 墨竹工卡 → 拉萨（约 270 km）

| 项目 | 说明 |
|------|------|
| 路况 | 翻越米拉山（318 最后一个大垭口），之后顺畅下坡至拉萨 |
| 海拔 | 3,440 m → 米拉山 5,013 m → 拉萨 3,650 m |
| 关键节点 | 米拉山隧道（已通车，垭口仍可选择走老路打卡）→ 墨竹工卡 → 达孜 → 拉萨 |
| 加油 | 沿途充足 |
| 住宿建议 | 拉萨市区 |
| 注意事项 | 抵达拉萨后优先做车辆终检；布达拉宫门票需提前预约 |

### Day 12–14 ｜ 拉萨休整 + 返程安排

| 项目 | 说明 |
|------|------|
| 安排 | 游览布达拉宫、大昭寺、八廓街；车辆保养或托运；规划返程（青藏线骑行 / 火车托运 / 物流发车） |
| 返程选项 | ① 青藏线骑回（加 5–7 天）② 火车托运摩托、人飞回 ③ 物流发车回 |

## 第三部分 · 应急预案

### 预案 A：高原反应应急

**预防措施**
- 出发前一周开始服用红景天（仅辅助），乙酰唑胺（Diamox）遵医嘱
- 每日饮水 ≥ 3L，避免饮酒和剧烈运动
- 行程设计遵循「阶梯上升」原则，不在 4,000 m 以上过夜超过必要天数

**轻度高反（头痛、恶心、食欲下降）**
1. 原地休息，不再上升海拔
2. 服用布洛芬缓解头痛，口服葡萄糖补充能量
3. 持续观察 2–4 小时，症状缓解后可低速继续

**中度高反（持续剧烈头痛、呕吐、呼吸困难）**
1. 立即停止骑行，患者由队友护送下降至少 500 m 海拔
2. 吸氧（随车携带便携氧气瓶或制氧袋）
3. 服用地塞米松（遵医嘱剂量）
4. 如 2 小时内无改善，联系最近县医院或拨打 120

**重度高反 / 高原肺水肿 / 脑水肿（意识模糊、口唇紫绀、呼吸急促带水泡音）**
1. 这是危及生命的紧急情况——立即呼叫救援
2. 用一切手段快速降低海拔（队友驾车载患者下山）
3. 持续吸氧，保持患者半坐位
4. 拨打 120 / 使用卫星通讯求助
5. 转运至最近有医疗条件的县城（芒康、八宿、波密、林芝均有县医院）

### 预案 B：车辆故障应急

**爆胎 / 漏气**
1. 安全靠边停车，打开双闪，后方 150 m 放置三角警示牌
2. 使用补胎工具修补（真空胎蘑菇钉/胶条）
3. 充气泵恢复胎压后低速骑至最近修理点复检
4. 无法修复时：联系当地拖车或将车拖至最近乡镇摩修店

**发动机 / 电路故障**
1. 检查保险丝、火花塞、电瓶接线
2. 简单故障现场排除
3. 复杂故障：将车推至安全位置，用对讲机通知队友
4. 联系品牌经销商远程指导（出发前存好厂家 400 热线）
5. 就近联系摩托修理店；康定、理塘、芒康、波密、林芝均有基本维修能力

**链条断裂（链传动车型）**
1. 使用备用链条接头接回
2. 调整链条松紧度后低速骑行
3. 无备件时寻求队友拖行至修理点（用拖车绳，速度不超过 20 km/h）

### 预案 C：交通事故应急

**处置流程**
1. 确保现场安全：双闪 + 警示牌 + 队友引导交通
2. 检查伤员状况，不要随意搬动疑似脊柱损伤者
3. 拨打 122（交警）和 120（急救），说明准确位置（手机定位截图）
4. 用手机拍照/录像保全证据（车辆位置、刹车痕迹、对方车辆信息）
5. 联系保险公司报案
6. 轻微事故可快速协商后继续行程；人员受伤必须等待救援

**队友受伤急救**
1. 止血：直接压迫法，使用急救包中的止血绷带
2. 骨折：用夹板/骑行护具临时固定，不要尝试复位
3. 意识丧失：侧卧位保持气道通畅，持续监测呼吸
4. 心跳呼吸骤停：CPR（30 次按压 : 2 次人工呼吸）直到专业救援到达

### 预案 D：恶劣天气与自然灾害

**暴雨 / 冰雹**
1. 就近找建筑物避让，不要在树下或山崖下停留
2. 等待天气好转再出发
3. 雨后注意路面积水、暗坑和落石

**塌方 / 泥石流（波密–通麦段高发）**
1. 发现前方塌方立即停车，切勿强行通过
2. 原路退至安全地带
3. 联系道班（路政养护部门电话提前存好）或等待机械清理
4. 可能需要绕行或在安全处等待数小时

**大雪 / 暗冰（高海拔垭口）**
1. 降低车速至 20 km/h 以下，用低档位控制速度
2. 安装防滑链（提前准备摩托车用简易防滑装置）
3. 跟随前车车辙行驶
4. 条件过于恶劣时在垭口前等待或折返

### 预案 E：人员走失 / 通讯中断

**预防**
- 每日出发前共享当天路线、住宿点、备选方案
- 蓝牙耳机保持开启，每 30 分钟报平安
- 手机离线地图标记每日必经的汇合点

**走失处置**
1. 原地等待 30 分钟，用对讲机呼叫
2. 在明显位置做标记（石头/反光条）
3. 如有信号，发送定位到微信群
4. 前方队友在最近加油站/检查站等待
5. 超过 2 小时无联络：启动卫星通讯定位或联系当地警方（110）

### 预案 F：物资短缺

**油料不足**
- 出发前计算各段里程与油箱续航，确保每段留有 50 km 余量
- 随车携带 2–3L 备用油桶（金属材质）
- 紧急情况下向过路车辆求助购油
- 高海拔段油耗增大约 15–20%，计算时预留裕量

**食品/饮水短缺**
- 每人随车携带 1 天应急口粮（压缩饼干、能量棒、牛肉干）和 1.5L 水
- 藏区河水不建议直接饮用，使用净水片处理
- 路过每个乡镇补充食品和饮用水

> 3 人队伍时，技术支援和医疗后勤合并为一人承担。

### 通讯规则

1. **主通讯**：蓝牙对讲耳机，保持常开
2. **备用通讯**：手持对讲机，统一频率，出发前测试
3. **紧急通讯**：卫星设备，仅在无信号区紧急情况使用
4. **每日两报**：早出发时互报检查完毕，晚抵达时确认全员到齐
5. **超车/停车口令**：统一使用简短口令（如"停车""靠边""注意对向"）
